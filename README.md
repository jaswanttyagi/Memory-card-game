#  Memory Match Game

A modern and responsive **Memory Card Matching Game** built with **React**. Test your memory by matching pairs of cards while keeping track of your moves and time. The game includes multiple difficulty levels, smooth card flip animations, and best score tracking using Local Storage.

## ✨ Features

- 🎮 Memory card matching gameplay
- 🃏 Randomly shuffled cards every game
- ⏱️ Live timer
- 🔢 Move counter
- 🏆 Win screen with final time and moves
- 🔄 Restart game anytime
- 📱 Fully responsive design
- 🎯 Three difficulty levels
  - Easy (4 × 4)
  - Medium (6 × 4)
  - Hard (6 × 6)
- 💾 Best score saved using Local Storage
- 🎨 Smooth flip animations
- 🌈 Modern glassmorphism UI

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/memory-match-game.git
```

Go to the project folder

```bash
cd memory-match-game
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

## 🎯 Game Rules

- Click a card to reveal it.
- Only two cards can be flipped at a time.
- Matching cards remain visible.
- Non-matching cards flip back after a short delay.
- Match all pairs to win.
- Try to finish with the fewest moves and in the shortest time.
