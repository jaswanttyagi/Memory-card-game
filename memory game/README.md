# Memory Match Game

A polished React memory card matching game with responsive design, difficulty levels, timer, move counter, and local best-score tracking.

## Features
- 4x4, 6x4, and 6x6 board options
- Random card shuffling on every new game
- Flip animation and match detection
- Move counter and timer
- Win modal with final stats
- Local storage best-score tracking
- Responsive design for desktop and mobile

## Tech Stack
- React
- Vite
- CSS

## How to Run Locally
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.
